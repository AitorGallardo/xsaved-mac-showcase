# Roadmap

The app is in TestFlight beta and under active development. Dates are intentions, not promises.

## Shipped

- A desktop library: an adaptive grid, full-text search, sidebar filters, and a detail inspector.
- Desktop interactions: a command palette, undo with a quiet toast, drag-and-drop into other apps, and a menu-bar companion.
- Background sync that keeps the library current and updates other devices on demand.
- A local copy of your bookmarks for an instant start and resilience to deleted tweets.
- Rich media — photos, video, and quoted tweets — in both the grid and the inspector.

## Next

- Deeper, more native cross-device sync with the iPhone app.
- Dark mode (the design system is already built for it).

## Later

- Truly instant cross-device updates rather than catching up on demand.
- Smarter browsing — finding bookmarks by meaning, not just words.

## Not planned

- A stretched-up phone layout. Every screen is built for the desktop.
- A second design language — new screens follow the shared one, in step with the iPhone app.
